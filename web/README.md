## Web

Powered By React and Redux.

### Includes

- React 16
- Redux
- Material-UI
- React-Snapshot
- Redux-Persist
- Prettier
- Husky
- ES-Next
- ESLint

### Running it

```
npm install
npm start
```

### Folder structure

    .
    ├── src
    │   ├── components          # Dumb components
    │   ├── config              # App configrations
    │   ├── constants
    │   ├── containers          # Smart components
    │   ├── redux               # Redux modules
    │   ├── utilities
    │   ├── App.js              # Root component
    │   ├── index.js
    └── package.json            # Node dependencies
